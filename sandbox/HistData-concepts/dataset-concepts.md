## Summary of Key Concept Categories

### Statistical Methods:
- regression, correlation, ANOVA, MANOVA
- t-test, significance test, hypothesis testing
- robust estimation, trimmed mean
- least squares, overdetermined systems
- logistic regression, odds ratios
- life tables, survival analysis
- sampling, quality control

### Graphical Methods:
- scatterplot, sunflower plot, pairs plot
- line graph, time series plot
- histogram, density plot, boxplot
- polar area diagram, coxcomb, rose diagram
- mosaic plot, balloonplot
- contour plot, 3D visualization
- choropleth map, dot map, flow map
- data ellipse, confidence ellipse
- biplot, screeplot

### Data Types:
- time series, longitudinal data
- spatial data, spatial-temporal
- bivariate/multivariate data
- frequency distribution, contingency table
- dyadic data, paired data

### Historical/Methodological:
- first statistical graph
- invention of methods (scatterplot, t-test, etc.)
- medical cartography, epidemiology
- quality control history
- robust methods development
- experimental design evolution
